# AgencyOS (OpenClaw Native Framework)

Este é o framework operacional do AgencyOS formatado NATIVAMENTE para o OpenClaw.

## Como instalar:
1. Copie o conteúdo desta pasta para a raiz do seu workspace OpenClaw.
2. Certifique-se de que o arquivo `agents.json` está na raiz (ele registrará todos os agentes operacionais da agência).
3. O Gateway utilizará o mapa de operações em `AGENTS.md`.

*Este repositório contém puramente a arquitetura de agentes, as 49 skills operacionais de agência, templates e workflows.*
