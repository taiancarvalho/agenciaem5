# Checklist Qualidade do Relatorio Mensal

**Cliente:** {nome}
**Periodo:** {mes/ano}

---

## Conteudo

- [ ] Resumo executivo claro e objetivo (max 3-5 linhas)
- [ ] Todos os canais ativos incluidos
- [ ] Dados do periodo anterior para comparacao
- [ ] Variacao percentual calculada
- [ ] Budget investido informado e correto

## Metricas

- [ ] KPIs principais vs meta definadas no plano
- [ ] CPL/CPA comparado com referencia saudavel
- [ ] ROAS ou retorno calculado (se vendas diretas)
- [ ] Criativos com melhor e pior desempenho identificados
- [ ] Campanhas com melhor resultado destacadas

## Insights

- [ ] Destaques do periodo (2-3 pontos positivos)
- [ ] Pontos de atencao identificados (se houver)
- [ ] Aprendizados registrados (o que funcionou, o que nao)

## Proximos Passos

- [ ] Acoes planejadas para o proximo periodo (2-3 itens)
- [ ] Responsavel e prazo indicado para cada acao
- [ ] Budget adjustments sugeridas (se necessario)

## Formatacao

- [ ] Nome do cliente correto
- [ ] Periodo correto
- [ ] Valores com formatacao brasileira (R$X.XXX,XX)
- [ ] Sem dados inconsistentes ou contraditorios
- [ ] Relatorio no formato adequado ao canal de envio

## QA do Relatorio

- [ ] Gestor de trafego revisou os dados
- [ ] CS validou coerencia geral
- [ ] Veredicto: pode enviar ao cliente
