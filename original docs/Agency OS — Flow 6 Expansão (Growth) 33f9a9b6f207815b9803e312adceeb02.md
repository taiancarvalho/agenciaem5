# Agency OS — Flow 6: Expansão (Growth)

# Objetivo

Definir como a agência escala resultados do cliente após validação inicial, aumentando receita, investimento e impacto de forma estruturada.

---

# Regra estrutural de logs

```
Este flow NÃO cria novos logs
Utiliza apenas:
- log-operacional.md
- log-performance-criativa.md
```

---

Definir como a agência escala resultados do cliente após validação inicial, aumentando receita, investimento e impacto de forma estruturada.

---

# Posição no sistema

```
Tráfego → gera leads
CRO → melhora conversão
Growth → escala o que funciona
```

---

# Papel do Growth

## Função principal

Identificar oportunidades de crescimento e escalar campanhas, canais e ofertas que já demonstraram resultado.

## O que ele faz

- analisa campanhas vencedoras
- aumenta investimento de forma controlada
- abre novos canais
- testa novas ofertas
- replica o que funciona
- elimina gargalos de crescimento

---

# Anti-papel

O Growth não deve:

- testar sem base
- escalar campanhas ruins
- ignorar dados
- agir antes de validação

---

# Regra principal

```
Só escala o que já provou funcionar
```

---

# Workflow macro

```
1. Identificar o que funciona
2. Validar consistência
3. Definir estratégia de escala
4. Executar expansão
5. Monitorar impacto
6. Ajustar e otimizar
7. Documentar aprendizado
```

---

# Etapa 1 — Identificar vencedores

## Fontes

- log de performance criativa
- campanhas ativas
- relatórios

## Critérios

- CPL consistente
- CPA saudável
- volume crescente

---

# Etapa 2 — Validação

## Perguntas

- esse resultado se mantém por 7 dias?
- por 14 dias?
- há estabilidade?

---

# Etapa 3 — Estratégia de escala

## Tipos

```
Vertical → aumentar verba
Horizontal → novos públicos
Criativo → novas variações
Canal → expandir plataforma
Oferta → testar novo produto
```

---

# Etapa 4 — Execução

- aumentar orçamento gradualmente
- duplicar campanhas
- abrir novos conjuntos
- lançar novas peças

---

# Etapa 5 — Monitoramento

- acompanhar métricas
- evitar queda de performance

---

# Etapa 6 — Ajuste

- reduzir se perder eficiência
- redistribuir verba

---
