## Estrategia

- [ ] Plano estrategico existe e esta aprovado
- [ ] Objetivo da campanha alinhado com plano
- [ ] Publico-alvo definido no plano esta configurado
- [ ] Budget aprovado pelo cliente

## Copy

- [ ] Copy escrita para cada peca
- [ ] Hook testavel (variante A/B disponivel)
- [ ] CTA presente e claro
- [ ] Sem erros ortograficos
- [ ] Validada por QA (*validar-copy)

## Criativo

- [ ] Criativos produzidos no formato do canal
- [ ] Dentro da paleta de branding
- [ ] Texto legivel com contraste adequado
- [ ] ID de versionamento aplicado
- [ ] Validado por QA (*validar-criativo)

## Tecnico

- [ ] Pixel instalado e testado
- [ ] Eventos de conversao configurados
- [ ] URL de destino funcionando
- [ ] Landing page testada (formulario/WhatsApp)
- [ ] Nomenclatura da campanha no padrao: [CLIENTE]_[OBJETIVO]_[CANAL]_[TIPO]_[ID]

## Campanha (na plataforma)

- [ ] Objetivo da campanha correto na plataforma
- [ ] Publico configurado (incluso/exclusoes)
- [ ] Budget definido
- [ ] Criativos anexados
- [ ] Copy preenchida em todos os campos
- [ ] Pelo menos 2 variações de criativo para teste

## Validacao Final

- [ ] QA validacao completa (*validar-campanha)
- [ ] Veredicto QA: **APROVADO**
- [ ] CS notificado sobre lancamento
