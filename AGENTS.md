# AgencyOS — Agentes Disponiveis

Ative um agente usando `@nome-do-agente` ou lendo o arquivo `.agencyos/agents/{nome}.md`.

| Comando | Agente | Persona | Quando Usar |
|---------|--------|---------|-------------|
| @agency-master | Atlas 🗺️ | Orquestrador | Setup do sistema, novo cliente, status geral |
| @cs | Sol ☀️ | Customer Success | Onboarding, setup tecnico, gestao do relacionamento |
| @estrategista | Vera 🎯 | Estrategista | Estrategia, posicionamento, plano de campanha |
| @gestor-trafego | Max 📊 | Gestor de Trafego | Campanhas pagas, otimizacao, relatorios |
| @copywriter | Cal ✍️ | Copywriter | Copy, angulos, adaptacao por canal |
| @designer | Lux 🎨 | Designer | Criativos, videos, UGC, layout visual |
| @qa-campanha | Zara 🔍 | QA | Validacao antes de publicar qualquer coisa |

## Commands Rapidos

### @agency-master
- `*new-client {nome}` — Criar workspace de novo cliente
- `*listar-clientes` — Listar todos os clientes
- `*status-cliente {nome}` — Status resumido de um cliente
- `*health-sistema` — Verificar integridade do sistema

### @cs
- `*iniciar-onboarding {nome}` — Iniciar onboarding
- `*coletar-briefing {nome}` — Coletar informacoes
- `*enviar-relatorio {nome}` — Enviar relatorio ao cliente

### @estrategista
- `*analisar-briefing {nome}` — Analisar briefing
- `*criar-plano-estrategico {nome}` — Criar plano

### @gestor-trafego
- `*estruturar-campanha {nome}` — Estruturar campanhas
- `*monitorar-performance {nome}` — Monitorar metricas
- `*modo-growth {nome}` — Ativar modo de escala

### @copywriter
- `*criar-angulos {nome}` — Criar angulos de copy
- `*escrever-copy {nome}` — Escrever copy para anuncios

### @designer
- `*gerar-imagem {nome}` — Produzir criativos visuais
- `*gerar-ugc {nome}` — Criar conteudo UGC

### @qa-campanha
- `*validar-campanha {nome}` — Validar antes de publicar
