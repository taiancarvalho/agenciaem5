# AgencyOS — Agentes Disponíveis

Ative um agente usando `@nome-do-agente` ou lendo o arquivo `.agencyos/agents/{nome}.md`.

| Comando | Agente | Persona | Quando Usar |
|---------|--------|---------|-------------|
| @agency-master | Atlas 🗺️ | CEO Estratégico | Estratégia do negócio, decisões de alto nível, OKRs, portfólio de clientes |
| @gerente | Marcos 🎯 | COO Tático | Executar operações multi-agente, orquestrar workflows, delegar tarefas |
| @cs | Sol ☀️ | Customer Success | Onboarding, setup técnico, gestão do relacionamento |
| @estrategista | Vera 🎯 | Estrategista | Estratégia, posicionamento, plano de campanha |
| @gestor-trafego | Max 📊 | Gestor de Tráfego | Campanhas pagas, otimização, relatórios |
| @copywriter | Cal ✍️ | Copywriter | Copy, ângulos, adaptação por canal |
| @designer | Lux 🎨 | Designer | Criativos, vídeos, UGC, layout visual |
| @qa-campanha | Zara 🔍 | QA | Validação antes de publicar qualquer coisa |

## Commands Rápidos

### @agency-master
- `*analise-negocio` — Revisão estratégica da agência
- `*analise-portfolio` — Visão do portfólio de clientes
- `*decisao {topico}` — Estruturar decisão de negócio
- `*new-client {nome}` — Criar workspace de novo cliente
- `*delegar-para-gerente {operacao} {cliente}` — Delegar operação para o COO

### @gerente
- `*executar-workflow {workflow} {cliente}` — Executar workflow completo
- `*analise-trafego {cliente}` — Orquestrar análise de tráfego
- `*lancar-campanha {cliente}` — Orquestrar lançamento de campanha
- `*onboarding {cliente}` — Orquestrar onboarding completo
- `*delegar {agente} {tarefa} {cliente}` — Delegar tarefa pontual
- `*status-operacao {cliente}` — Status das operações em andamento

### @cs
- `*iniciar-onboarding {cliente}` — Iniciar onboarding
- `*coletar-briefing {cliente}` — Coletar informações
- `*enviar-relatorio {cliente}` — Enviar relatório ao cliente

### @estrategista
- `*analisar-briefing {cliente}` — Analisar briefing
- `*criar-plano-estrategico {cliente}` — Criar plano

### @gestor-trafego
- `*auditar-conta {cliente} meta` — Auditoria da conta
- `*subir-estrutura-inicial {cliente}` — Subir campanhas
- `*monitorar {cliente}` — Monitorar métricas
- `*modo-growth {cliente}` — Ativar modo de escala

### @copywriter
- `*criar-angulos {cliente}` — Criar ângulos de copy
- `*escrever-copy {cliente}` — Escrever copy para anúncios

### @designer
- `*gerar-imagem {cliente}` — Produzir criativos visuais
- `*gerar-ugc {cliente}` — Criar conteúdo UGC

### @qa-campanha
- `*validar-campanha {cliente}` — Validar antes de publicar
