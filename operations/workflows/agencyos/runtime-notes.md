# Runtime Notes

## Como operar no OpenClaw
- ISIS/main atua como coordenadora.
- Cada agente AgencyOS recebe uma task objetiva com cliente, inputs e output esperado.
- O agente devolve artefato, não brainstorm infinito.
- Toda troca relevante entre agentes gera handoff em arquivo.
- QA da Zara é bloqueante.
