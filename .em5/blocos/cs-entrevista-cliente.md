# Bloco: Entrevista Consultiva Cliente

## O que faz
Conduz call/conversa estruturada (NÃO questionário). Tom consultivo.

## Inputs
- contexto cliente (perfil OR onboarding OR situação atual)
- objetivo entrevista (board / perfil / QBR / retenção)
- pauta específica (perguntas-âncora)

## Execução
1. Abertura: contextualizar duração + objetivo
2. 1 seção por vez (não despejar 20 perguntas)
3. Validar entendimento antes avançar
4. Anotar respostas + observações não-ditas (tom, objeções implícitas)
5. Fechar: próximos passos claros

## Output
- markdown estruturado respostas + observações
- Anotações internas (objeções não-ditas, sinais de churn, oportunidades)

## Anti-padrões
- ❌ Questionário formulário
- ❌ Pressionar resposta
- ❌ Ignorar linguagem corporal/tom
