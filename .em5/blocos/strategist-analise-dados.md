# Bloco: Análise Estratégica de Dados

## O que faz
@strategist analisa dados crus → insights acionáveis + hipóteses.

## Inputs
- dados (auditoria.md OR relatórios OR métricas)
- plano-estratégico (KPIs alvo + ICP)
- histórico (3 últimas notas-sessão)

## Execução
1. Ler dados + plano
2. Identificar gaps (resultado vs meta)
3. Mapear ângulos vencedores/perdedores
4. Formular 3-5 hipóteses testáveis (Se X, então Y, porque Z)
5. Recomendar próximas ações priorizadas

## Output
- Análise estruturada (markdown)
- Hipóteses + recomendações

## Anti-padrões
- ❌ Análise sem dado (opinião)
- ❌ Hipóteses não-testáveis
- ❌ 10+ recomendações (paralisia — max 5)
