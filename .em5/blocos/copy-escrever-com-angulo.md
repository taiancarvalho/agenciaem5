# Bloco: Escrever Copy com Ângulo Definido

## O que faz
Escreve copy (ad/post/email/blog/roteiro) usando 1 dos 5 ângulos (DOR/DESEJO/PROVA/CONTRAINTUITIVO/STATUS).

## Inputs
- briefing (objetivo + público + plataforma + ângulo escolhido)
- ICP detalhado
- branding/voz cliente
- exemplos peças que performaram

## Execução
1. Ler `clientes/{nome}/copy/analise-icp.md`
2. Aplicar ângulo dominante + 1-2 secundários
3. Estrutura por canal:
   - Ad: hook + amplificação + solução + CTA
   - Post: hook + desenvolvimento + CTA específico
   - Email: subject + preview + corpo + 1 CTA + PS
   - Blog: H1 keyword + outline + texto SEO
   - Reel: hook 3s + dev + CTA loop
4. Versionar CR-XXX.md
5. Salvar `clientes/{nome}/copy/`

## Output
- copy.md + ângulo declarado
- Versionado CR-XXX

## Anti-padrões
- ❌ Misturar 5 ângulos
- ❌ Hook genérico ("quer vender mais?")
- ❌ CTA vago ("saiba mais")
- ❌ Sem ICP lido
