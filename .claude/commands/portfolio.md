# /portfolio — Análise Portfólio Agência (estratégico)

**Argumento:** `/portfolio` (sem args — varre todos clientes)

## Classificação
**Estratégico** → @ceo

## Roteamento

```
@ceo Atlas — operação: análise portfólio
Receita: analise-portfolio
Cadência: mensal (dia 5)
Referência: .em5/receitas/analise-portfolio.md
```

## O que vê

- MRR · ARR · Churn · NRR · LTV/CAC
- Tier A/B/C clientes
- Pipeline comercial
- Capacidade operacional
- Decisões pendentes (crítico/atenção/OK)

## Output

`clientes/_agencia/portfolio/dashboard-{YYYY-MM}.md` (usa template `system/templates/operacao/dashboard-portfolio.md`)
