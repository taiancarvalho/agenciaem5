# /setup — Configurar o em5

Leia `.em5/setup/system-onboarding.md` e execute o wizard de configuração passo a passo.

## O que este comando faz

1. Coleta nome da agência, nome do dono, serviços oferecidos
2. Configura nomes fantasia dos 8 agentes (ou usa os defaults)
3. Configura Composio API Key e integrações
4. Gera `em5-config.yaml` com toda a configuração
5. Atualiza `.claude/CLAUDE.md` com os dados da agência
6. Confirma que todos os 8 agentes estão registrados em `.claude/agents/`

## Quando usar

- Primeira vez instalando o em5
- Para reconfigurar nome de agente, serviços ou integrações
- Após adicionar nova integração no Composio

## Instrução

Inicie o wizard agora. Pergunte item por item, não tudo de uma vez.
Ao final, gere os arquivos de configuração e confirme o que foi criado.
